READ ME 

sudoku.py exhibits the backtrack algorithm for solving a sudoku puzzle in a text based format.
  Additionally contains the methods for generating a solvable sudoku puzzle
  
sudoku_GUI.py shows how the backtracking algorithm works through solving the puzzle using a GUI.
  Simply run the code and hit space bar to solve.
